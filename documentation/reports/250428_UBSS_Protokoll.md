## game flow

### Konstantin

Haupt "Ressource" Zufriedenheit/Nahrung out of 100
	- Resultiert in Nachteilen falls diese in einem geringen Bereich liegt
Anfang: Tutorial 
- erste Kompartimente platzieren für Nahrungsmittelproduktion
- erste Verarbeitungsstätten für Nahrungsmittel platzieren
- erster Supermarkt für die Nahrungsverteilung an die Bewohner wird platziert

Über die Zeit brechen klassische Methoden der Nahrungsmittelproduktion immer mehr ein
-> Nahrungsmittelproduktion muss in den Urbanen Bio Räumen immer weiter hochgefahren werden

Transportwege müssen zwischen den Gebäuden gebaut werden / optimiert werden

Kompartimente, Facilities, Transportwege können geupgraded werden

Anforderungen der Bewohner an Ernährung muss erfüllt werden

Ziel des Spiels ist es so viele Tage wie möglich die Bevölkerung zu ernähren,
sobald Zufriedenheit/Nahrung auf 0 fällt ist das Spiel vorbei und man erhält
seinen erreichten Score

### Harvey

* limitierte Ressourcen (Geld, Essen, etc.)
* Geld o.ä. benutzen um z.B. Gebäude zu Bauen / zu enhancen
* Essen verschwendet sich pro Runde
* jede Runde chance neue Häuser zu spawnen
* Häuser bringen Population
* mehr Population = mehr Essensverbrauch
* jede Runde chance, dass Häuser von alleine sich enhancen in größere Häuser -> mehr population
* Essen kann durch Landwirtschaft / die Erwähnten Nahrungsmittel gewonnen werden
* Grundproduktion durch schon bestehende Landwirtschaft
* zeitliche Ersetzung von alten Produktion und Ruinen durch neue urbane Lebensmittelproduktion
* gewisser Zeitdruck durch random event wie z.B. einschränkung von nutzbarem Landwirtschaftsraum (no land)
* gewisse Gebäude können durch genug enhancement “autonom” im Sinne von Nahrung werden und produzieren für sich alleine genug Nahrung
* ansonsten muss Umgebung genug Verteilung (Supermärkte) haben

* verbindung von Produktion, Lieferung und Verteilung durch “Dronenlinien”
* realtime aber pausieren super

### Alexander J

Spielkonzept


Ziel: bestimmte Anzahl an Bevölkerung zu ernähren (ohne Import/Export)
Endless: Bevölkerung wächst immer mehr


Verlieren: wenn Bevölkerung für x-tage nicht ausreichend ernährt wurde


* Stadt wird vom Spiel vorgegeben
* Wenn man gewisse Meilensteine erreicht (x-Bevölkerung für y-tage ernährt) wächst die Stadt -> mehr Häuser, mehr freie bebaubare Stellen in der Stadt für Produktion (random zugewiesen)
* Stadt steht fest/ wird vom Game vorgegeben
* neue Stadtteile werden nach und nach aufgedeckt
* Nährwert Haushalt der Bevölkerung muss gedeckt sein (Kalorien, Kohlenhydrate, Eiweiß, Fett)


Factory:
* Erzeuger (Algen-/Insektenfarmen; Wasseraufbereitung)
* Weiterverarbeitung (Biomass -> Essen Produkte)
* Endstation = Supermarkt
* Unterschiedliche Endprodukte haben unterschiedliche Nährwerte


Inside Factory:
* Kleiner Raum auf Grid
* Kleine Maschinen können platziert werden
* Produkte werden von Maschine zu Maschine gebracht (mithilfe von zB keinen Robotern/ zusammenstellen der Maschinen)
* Mehrere unterschiedliche Maschinen die synergieren (perfekt ratio)
* Sicht auf endlosmodus: Maschinen kann man upgraden


Transport mithilfe von Drohnen (evtl sind diese erst limitiert, und später kann man mehr dazu kaufen)
Transportkette: Erzeuger -> Weiterverarbeitung A (-> B) -> Supermarkt


Drohnen haben Batterien und können nur eine gewisse Distanz fliegen (Schwierigkeit von Logistikketten)
Transport ist möglich zu upgraden (am anfang langsam, dann viel mehr)


Supermarkt hat 3 Regale (für die jeweiligen Kategorien von Essen; Algen, Quallen, Insekten)

### Tobias

man startet das spiel und sieht eine stadt mit verschiedenen haeusern und fabriken. Die fabriken sehen schmutzig aus und die haeuser etwas altmodisch aber normal.
Es oeffnet sich ein fenster wo man erklaert bekommt, dass die jetzige oekonomie in STADTNAME ausgeglichen ist, aber die leute einengt, die umwelt zerstoert und nicht zukunftsfaehig ist.
Das fenster schliesst sich durch klick auf OK.
Man wird als naechstes darauf hingewiesen, dass eine bestimmte fabrik verschwinden wird um platz fuer hauser zu schaffen.
Die Fabrik verschwindet, es erscheinen neue heauser an deren stelle.
Es gibt am unteren rand mehrere schaltflaechen. die rechte mit einem pfeil nach oben symbol leuchtet auf. Der rest des bildschirms wird leicht dunkler.
klick auf die schaltflaeche.
Es oeffnet sich ein fenster, dort ist alles ausgegraut ausser ein "upgrade basic home" item.
klick auf item
Fenster schliesst sich, das item folgt der maus. Es wird ueberall rot hinterlegt, ausser auf den hausern.
(hilfsfenste am rand, es sagt das man die haeuser mit dem "upgrade basic home" anklicken soll.)
klick auf die haeuser, sie ersetzen damit die funktionalitaet der fabrik.
Allerdings reicht es nicht ganz. Man muss also noch andere haueser anklicken, um den haushalt auszugleichen.
Wenn man das nicht tut, wird man auf das haushalt/energie menu hingewiesen. Es blinkt auf, wenn eine resource nicht genug vorhanden ist.
